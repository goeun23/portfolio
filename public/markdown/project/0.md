- 실시간 AI 공감 상담 : 사용자의 일기와 감정 태그를 분석하여 Gemini 2.0 AI 응답 대기 시간을 최소화하는 스트리밍 UI 적용으로 사용자 경험 개선
- 서버리스 백엔드 구축 (Supabase) : Next.js Middleware와 Supabase Auth를 연동한 보안 세션 관리 및 구글 로그인 구현
- RSC 기반 성능 최적화 : 기존 컴포넌트를 서버 컴포넌트로 전환하여 초기 데이터 프리페치(Pre-fetch) 구현
- 검색 엔진 최적화 (SEO) : 페이지별 동적 `Metadata` 및 소셜 공유를 위한 `Open Graph` 태그 적용
- (Refactoring) 전역 상태 관리 경량화 
    - 기존 Redux Toolkit을 제거하고 React Context API로 마이그레이션
    - 외부 의존성을 줄여 번들 사이즈를 최적화하고 유지보수 복잡도 감소
