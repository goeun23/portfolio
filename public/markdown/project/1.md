__Server Actions 기반 아키텍처 리팩토링__
- 기존 REST API의 복잡한 보일러플레이트를 직관적인 함수 호출(RPC) 방식으로 대체
- API Key 및 DB 접속 정보를 서버 환경에 격리, 클라이언트 노출 방지
- 비즈니스 로직(Supabase, 공공데이터 처리)을 Service Layer로 분리 및 모듈화
- XML 파싱을 서버로 위임하고, Hybrid Rendering을 통해 클라이언트 부하 최소화


__Core Web Vitals 개선__
- 중복 API 호출 방지를 위한 캐싱(Caching) 전략 도입으로 동일 데이터 조회 속도 단축
- React Query와 Intersection Observer를 결합한 가상화 무한 스크롤 구현
- 이미지 업로드 시 병렬 처리 및 Optimistic Update로 대기 시간 제로화
- LCP 3.2s → 0.8s 성능 개선
