- Next.js App Router 아키텍처: 서버/클라이언트 컴포넌트 분리를 통한 렌더링 최적화(LCP 개선) 및 SEO 친화적 구조 설계
- Server Actions 도입: 기존 API Route를 제거하고 백엔드 로직을 직접 호출하여 구조 단순화 및 Type-safety 확보
- Supabase 기반 이미지 처리: 다중 이미지 병렬 업로드(Parallel Uploads) 로직을 구현하여 미디어 처리 로직 구현
- 무한 스크롤(Infinite Scroll) 구현: Intersection Observer와 React Query를 결합하여 데이터 리스트 렌더링 성능 최적화
- SEO(검색 엔진 최적화): 동적 메타 태그(Open Graph Protocol) 적용으로 소셜 미디어 공유 가시성 및 검색 노출 최적화
